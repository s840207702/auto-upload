# Social Auto Upload 新手一键包

这是基于 `zydgmail/social-auto-upload` 的本地一键包底座，用来快速得到一个能看、能用、能基本跑通的多平台发布系统。

## 启动

双击：

```text
start-oneclick.bat
```

启动后浏览器会打开：

```text
http://127.0.0.1:5409/
```

## 当前定位

- 这是 MVP 底座，不是最终商业级产品。
- 当前优先解决：新手能启动、能看到 Web 管理界面、能添加账号、能上传素材、能进入发布流程。
- 后续再逐步修平台适配、登录稳定性、发布失败恢复和打包体积。

## 推荐使用流程

1. 打开 `账号管理`。
2. 添加一个测试账号。
3. 扫码登录并确认账号状态正常。
4. 打开 `素材管理` 上传一个短视频。
5. 打开 `发布中心`，选择账号和素材。
6. 填写标题、标签和定时信息。
7. 先用一个平台测试发布，不要一开始就批量多平台。

## 数据位置

账号记录和素材记录会写入：

```text
db/database.db
```

登录态文件会写入：

```text
cookiesFile/
```

上传素材会写入：

```text
videoFile/
```

## 当前已知风险

- 国内平台页面经常变，选择器可能需要维护。
- 浏览器自动化登录比官方 API 更脆弱。
- 多账号批量发布需要先小规模验证。
- B 站、视频号、抖音等平台可能有额外风控或验证码。
- 如果账号添加一直停在“请求中”，先关闭启动窗口并重新双击启动脚本。新版启动逻辑会使用系统 Chrome/Edge 或 Playwright 默认 Chromium，不再依赖缺失的 `third_party` 浏览器目录。
