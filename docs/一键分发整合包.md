# 一键分发整合包

这个仓库可以生成给普通用户使用的 Windows 整合包。用户电脑不需要提前安装 Python、Node、npm 或 Playwright。

## 构建命令

在项目根目录运行：

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\build-oneclick-package.ps1
```

构建完成后会在 `release/` 下生成：

- `auto-upload-oneclick-时间戳/`：可直接运行的文件夹。
- `auto-upload-oneclick-时间戳.zip`：可分发压缩包。

## 用户使用方式

用户解压 zip 后，双击根目录的 `start-oneclick.bat` 即可。启动脚本会打开本地页面 `http://127.0.0.1:5409/`。包内同时包含 `README-zh-CN.txt`，可直接发给新手用户阅读。

外层文件夹可以改成中文名，但不要改包内固定目录名，例如 `runtime/`、`frontend/`、`cookiesFile/`、`db/`、`logs/`、`avatars/`、`videoFile/`。

## 安全边界

构建脚本只复制源码、前端产物和运行时依赖，不会复制这些本地敏感数据：

- `cookiesFile/`
- `db/`
- `logs/`
- `avatars/`
- `videoFile/`
- `.venv/`
- `.git/`
- `frontend/node_modules/`

分发前仍建议打开生成目录确认以上运行数据目录为空。
